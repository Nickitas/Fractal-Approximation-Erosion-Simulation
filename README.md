## 🎯 Научная цель

Разработка и верификация комплексной математико-алгоритмической модели геометрического образа береговой линии, сочетающей фрактальные свойства и динамику изменения под воздействием эрозионных процессов.

---

## 🚀 Возможности

- Загрузка береговой линии из JSON-файла
- Расчёт длины полилинии по географическим координатам
- Демонстрация парадокса береговой линии
- Фрактальная аппроксимация кривой Коха с управляемым числом итераций
- Приближённый расчёт фрактальной размерности методом box-counting
- Генерация SVG для исходной береговой линии и серии `koch_iter_0.svg ... koch_iter_N.svg`
- CLI с подкомандами: `all`, `coastline`, `paradox`, `koch`, `dimension`

---

## 🧩 CLI

Подкоманды CLI:

- `fraes all` — запускает все текущие демонстрации подряд
- `fraes coastline` — считает метрики исходной береговой линии и сохраняет `coastline.svg`
- `fraes paradox` — показывает рост длины при уточнении полилинии
- `fraes koch` — строит кривую Коха и сохраняет серию `koch_iter_0.svg ... koch_iter_N.svg`
- `fraes dimension` — считает фрактальную размерность на итерациях Коха и тоже сохраняет серию SVG

Флаги подкоманд:

- `--input` — путь к JSON с точками береговой линии
- `--iterations` — максимальное число итераций Коха
- `--output` — путь к одному SVG или к директории с артефактами

Поведение `--output`:

- если флаг не указан, файлы сохраняются в `./output/`
- если путь заканчивается на `.svg`, это одиночный SVG-файл
- если путь не заканчивается на `.svg`, это директория для результатов
- для команд `koch`, `dimension` и `all` удобнее передавать именно директорию

## 🛠 Установка и запуск

```bash
# Клонирование
git clone https://github.com/Nickitas/Fractal-Approximation-Erosion-Simulation.git
cd Fractal-Approximation-Erosion-Simulation

# Сборка
go build -o fraes .

# Список команд
./fraes --help

# Базовый запуск
./fraes coastline
```

---

## ⚡ Примеры использования

```bash
# 1. Метрики исходной береговой линии + SVG в ./output/
./fraes coastline

# 2. Кривая Коха с 4 итерациями и серией SVG в директории
./fraes koch --iterations 4 --output ./output/koch

# 3. Расчёт размерности с альтернативным JSON-файлом
./fraes dimension --iterations 6 --input data/black-sea.json

# 4. Полный текущий прогон со всеми демонстрациями
./fraes all --output ./output/full-run
```

## 📊 Выходные данные

После выполнения в каталоге `--output` появятся:

- `coastline.svg` — визуализация исходной береговой линии
- `koch_iter_0.svg ... koch_iter_N.svg` — серия SVG для итераций кривой Коха

Сейчас проект не генерирует `gif`, `csv` или `json`-отчёты. Это следующие этапы из плана разработки.
