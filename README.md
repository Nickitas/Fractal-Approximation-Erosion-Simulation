# FRAES (Fractal Approximation & Erosion Simulation)
 
Модуль реализации фрактальной аппроксимации и симуляции эрозии береговой линии прибрежной системы. 

**Описание модуля:** 
Проект посвящен геометрическому моделированию прибрежных систем Черного моря с использованием фрактальной геометрии. Береговые линии часто имеют фрактальную структуру (самоподобие на разных масштабах), что делает их идеальным объектом для изучения через кривые вроде Коха или L-системы. 

Модуль представляет из себя программу, которая генерирует SVG-визуализацию, симулирует изменения берега со временем и вычисляет математические характеристики. Это сочетает геометрию, вероятностные модели и вычисления, с практическим применением к экологии (моделирование эрозии для прогнозирования изменений прибрежных зон).
