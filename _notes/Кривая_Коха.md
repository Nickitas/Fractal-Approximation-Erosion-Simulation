# Кривая Коха 

**Кривая Коха** — это один из самых красивых и важных фракталов в истории математики.  
Её придумала в 1904 году шведский математик Хельге фон Кох, чтобы показать: **можно построить непрерывную кривую, у которой ни в одной точке нет касательной** — то есть она нигде не гладкая!

### Как строится кривая Коха (снежинка Коха)

Начинаем с прямого отрезка (итерация 0):

```
──────
```

**Итерация 1**: каждый отрезок заменяем на 4 отрезка длиной в 1/3, образуя «шип» (равносторонний треугольник посередине):

```
        /\
       /  \
______/    \______
```

**Итерация 2**: то же самое делаем с каждым из четырёх новых отрезков → получаем 16 отрезков:

```
     /\    /\
    /  \  /  \
___/    \/    \___
```

**Итерация 3** → 64 отрезка, **итерация 4** → 256, и так далее.

Полная замкнутая фигура из трёх таких кривых называется **снежинка Коха**.

### Главные математические свойства

| Параметр                     | Значение                          | Что это значит |
|------------------------------|-----------------------------------|----------------|
| Количество отрезков          | Nₙ = 4ⁿ                           | растёт экспоненциально |
| Длина одного отрезка         | Lₙ = (1/3)ⁿ × L₀                 | уменьшается в 3สอบ 3 раза |
| Общая длина на итерации n    | Lₙ = L₀ × (4/3)ⁿ                  | **растёт бесконечно!** |
| Площадь снежинки             | конечная (≈ 1.6 × площадь исходного треугольника) | ограничена |
| Фрактальная размерность D    | log(4)/log(3) ≈ **1.2618595**     | между линией (D=1) и плоскостью (D=2) |
| Дифференцируема?             | **ни в одной точке**              | абсолютно «шершавая» |

### Почему это революционно

До Коха все думали, что:
- если кривая **непрерывна** → у неё почти везде есть касательная
- если **длина конечна** → она «гладкая»

Кох показал: можно иметь **непрерывную кривую бесконечной длины**, которая **нигде не гладкая**.  
Это стало одним из первых примеров «патологического» объекта, который разрушил интуицию XIX века и подготовил почву для фрактальной геометрии Мандельброта.

### Кривая Коха и парадокс береговой линии

Именно кривая Коха — идеальная модель того, почему длина береговой линии растёт при увеличении детализации:

| Уровень детализации | Аналогия с природой                     |
|---------------------|-----------------------------------------|
| 0                   | карта 1:10 000 000                      |
| 1                   | карта 1:1 000 000                       |
| 2–3                 | спутниковые снимки                      |
| 4–6                 | измерения на местности, каждый камень  |
| → ∞                 | длина → ∞ (как у настоящих берегов)     |

### Вариации кривой Коха (очень красивые!)

| Название                  | Угол «шипа» | Размерность D      | Вид                     |
|---------------------------|-------------|--------------------|-------------------------|
| Классическая Коха         | 60°         | ≈ 1.2619           | снежинка                |
| Квадратная Коха           | 90°         | ≈ 1.5              | похожа на лабиринт      |
| Minkowski sausage         | 90°         | 1.5                | «колбаса»               |
| Коха с случайным углом    | 50–70°      | ≈ 1.1–1.3          | **самая реалистичная модель берега** |

### Что будет, если применить Коха к Чёрному морю (как мы сделали)

- Исходная длина: ≈ 4500 км
- После 1 итерации: ×4/3 ≈ 6000 км
- После 2: ×16/9 ≈ 8000 км
- После 5: ≈ 17 000 км
- После 10: ≈ 137 000 км
- После 20: ≈ 9 миллионов км (уже 225 раз вокруг Земли!)

А вся кривая при этом **по-прежнему лежит внутри узкой полосы** вдоль реального берега.

> Кривая Коха — это простейший способ превратить обычную линию в бесконечно длинную, нигде не гладкую, но ограниченную по площади структуру — идеальный математический прообраз реальной береговой линии.
